## Daily ritual

1. Open this table. Find today's row.
2. Read the pattern explanation in the phase page before coding.
3. Solve each listed problem WITHOUT looking at the solution first (give it 20-25 min).
4. If stuck: look at the pattern hint only, not the full solution. Try again.
5. After solving (or after seeing the solution): re-type the solution from scratch without copying.
6. Log in tracker: Problems solved, Solved without hints, Time taken, Key insight.
7. Any problem you couldn't solve cold goes in **Problems to revisit** — redo it in 3 days.
